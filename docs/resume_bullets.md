# Resume Bullets

Use one of these versions depending on space.

## Three-Bullet Version

- Built an end-to-end LLM-powered academic literature mining pipeline that retrieves arXiv papers, parses PDFs, extracts structured research metadata, and constructs an interactive knowledge graph.
- Implemented schema-validated LLM extraction with retry logic, targeted re-extraction, quality warnings, failed-paper reporting, and support for OpenRouter/Gemini providers plus mock-mode testing.
- Developed a Streamlit dashboard and NetworkX/PyVis graph exports to explore relationships among papers, methods, datasets, baselines, metrics, and research keywords.

## Compact Version

- Built an LLM-powered academic literature knowledge graph pipeline with arXiv retrieval, PDF parsing, schema-validated extraction, quality reporting, and a Streamlit exploration dashboard.

## Interview Summary

This project turns unstructured academic PDFs into a structured research database and graph. The pipeline retrieves arXiv papers, parses and cleans PDF text, extracts fields such as research problem, method, datasets, baselines, metrics, limitations, and keywords with LLM providers, validates the output, then builds graph exports and an interactive Streamlit demo.
